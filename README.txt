========================================
= IP Messenger meets Growl for Windows =
========================================

* This is derivertive IP Messenger client that enabled "growling" feature.

* IP Messenger would "growl" in case of "Non popup" notification mode.

* This software based on IP Messager for Win 2.06.
  * URL: http://www.ipmsg.org/index.html

* Please refer original documents placed in \ORIGINAL-DOCS .

* Building environments:
  * Visual C++ 6.0 
  * Windows XP
  * Growl for Windows (Require for operational environment)
